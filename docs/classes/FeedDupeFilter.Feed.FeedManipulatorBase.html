    xmlErrorHandler(int $code, string $message) : false | void

    This handler is set and unset in loadFeed() to catch errors which occurred during parsing of the XML document in DOMDocument. The error message is extracted from the error string by using an regular expression and stored in the loadingError() field.

    see \FeedDupeFilter\Feed\set_error_handler()
